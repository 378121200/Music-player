<h1 align="center">Welcome to Music-player 👋</h1>

## 🚀 如何运行

> node 版本 `[6.4.0]`

### ✨开发过程

```
# 克隆
git clone https://github.com/powerdong/Music-player.git
```

```
# 打开项目目录
cd Music-player

cd musicPlayer
```

```
# 安装依赖
npm install
```

```
# 开启本地服务运行项目
npm run dev
```
